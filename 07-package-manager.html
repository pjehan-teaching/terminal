Gestionnaire de paquets

Présentation

Un gestionnaire de paquets (package manager) est un ensemble d'outils permettant de simplifier l'installation, la mise à jour et la suppression de programmes.

La commande apt est utilisée pour utiliser le gestionnaire de packets.

Installation

La commande apt install [package-name] permet d'installer un programme.

Le gestionnaire de paquets va alors rechercher le programme dans la liste des dépôts disponibles.

Il est possible d'ajouter un nouveau dépôt avec la commande suivante :

add-apt-repository [repository-name]
apt update

La liste des dépôts est sauvegardée dans le fichier suivant :

vim /etc/apt/sources.list
Ne pas ajouter des dépôts de source inconnues !

Mise à jour

Pour mettre à jour les paquets installés, vous pouvez utiliser la commande apt upgrade

Suppression

Pour supprimer un paquet, vous pouvez utiliser la commande apt remove [package-name]

Pour supprimer un paquet et ses fichiers de configuration associés, vous pouvez utiliser la commande apt purge [package-name]

Pour supprimer les paquets qui ne sont plus utiles (car installé comme dépendances), vous pouvez utiliser la commande apt autoremove
