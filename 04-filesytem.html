Manipuler les fichiers et dossiers

Manipuler les dossiers

Pour créer un dossier, vous pouvez utiliser la commande mkdir.

Pour le supprimer, vous pouvez utiliser la commande rmdir.

Cependant, la commande rmdir ne permet pas de supprimer un dossier contenant des fichiers. Dans ce cas, vous pouvez utiliser la commande rm -r pour supprimer de façon récursive.

Utilisez la commande rm avec beaucoup de précaution !

Manipuler les fichiers

Pour créer un fichier, vous pouvez utiliser la commande touch.

Pour le supprimer, vous pouvez utiliser la commande rm.

Pour le déplacer, vous pouvez utiliser la commande mv :

mkdir public_html
touch index.html
mv index.html public_html/index.php
La commande mv permet également de renommer un fichier

Manipuler les fichiers

Pour copier un fichier, vous pouvez utiliser la commande cp.
