VIM (Vi IMproved)

Présentation de VIM

VIM est un éditeur de texte open source permettant de modifier des fichiers directement dans le terminal.

Les modes de VIM

VIM dispose de plusieurs modes. Les 3 modes principaux sont :

  • Normal ou commande
  • Insertion
  • Ligne de commande

Ouvrir un fichier

Vous pouvez ouvrir un fichier avec la commande suivante :

vim index.html
Si le fichier n'existe pas, il sera créé

Quitter VIM

Pour quitter VIM, utilisez la commande :q.

Vous ne pouvez pas quitter un fichier si vous avez effectué des modifications. Pour forcer à quitter, saisissez la commande :q! (Les modifications ne seront pas enregistrées).

Pour sauvegarder et quitter, utilisez la commande :x.

Mode normal

En mode normal, lorsque vous appuyez sur des touches de votre clavier, vous exécutez des commandes.

Par exemple, si vous appuyez 2 fois sur la touche d de votre clavier, cela va supprimer la ligne sur laquelle se trouve votre curseur.

Appuyez 2 fois sur la touche y pour copier une ligne puis sur la touche p pour coller.

Mode normal

Appuyez sur la touche 0 pour aller en début de ligne.

Appuyez sur la touche $ pour aller en fin de ligne.

Appuyez sur les touches g g pour aller au début du fichier.

Appuyez sur la touche G pour aller à la fin du fichier.

Mode normal

Appuyez sur la touche u (undo) pour annuler le dernier changement.

Appuyez sur les touches Ctrl + r (redo) pour annuler le dernier changement.

Appuyez sur la touche . pour répéter le dernier changement.

Mode insertion

Appuyez sur la touche i pour passer en mode insertion.

Vous pouvez alors saisir du texte comme dans un éditeur classique.

Mode ligne de commande

Le mode ligne de commande permet d'exécuter des commandes sur un fichier.

Il peut par exemple être utilisé pour rechercher dans un fichier en appuyant sur la touche / depuis le mode normal puis en saisissant le texte recherché.

Le mode ligne de commande est utilisé pour quitter VIM

Modifier la configuration

Vous pouvez modifier la configuration de VIM depuis le mode "ligne de commande" en saisissant par exemple :

set number

Vous pouvez également enregistrer votre configuration en créant un fichier ".vimrc" dans votre dossier home :

vim ~/.vimrc
