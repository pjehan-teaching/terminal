Parcourir le disque dur

Afficher l'aide d'une commande

Le terminal vous permet de saisir des commandes afin d'effectuer des actions.

Pour savoir comment utiliser une commande et connaître les différentes options disponibles, vous pouvez ensuite utiliser la commande man suivit du nom de la commande que vous souhaitez utiliser :

man pwd

man ouvre la documentation de la commande dans le terminal. Vous pouvez appuyer sur la touche q pour quitter.

Vous pouvez également ajouter le paramètre --help ou -h sur n'importe quelle commande pour afficher l'aide.

Le paramètre -h ne fonctionne pas avec toutes les commandes

Nettoyer le terminal

Vous pouvez nettoyer le terminal avec la commande clear.

Remonter dans l'historique des commandes

Vous pouvez remonter dans l'historique des commandes en utilisant les touches ⬆ et ⬇.

Vous pouvez également effectuer une recherche en utilisant la combinaison de touches Ctrl + r. Tapez ensuite la commande recherchée puis utilisez à nouveau la combinaison de touches Ctrl + r pour parcourir les résultats de recherche.

Ne pas saisir de mot de passe en clair dans le terminal !

Se situer

Lorsque vous ouvrez le terminal, vous vous trouvez par défaut dans le dossier home (identifié par le caractère "~") de l'utilisateur.

A tout moment, vous pouvez afficher le chemin complet où vous vous situez en utilisant la commande pwd.

Lister le contenu d'un dossier

Pour lister le contenu d'un dossier, vous pouvez utiliser la commande ls.

Cette commande affiche les différents fichiers et dossiers directement dans la console. Le code couleur suivant permet de rapidement identifier le type des fichiers :

Couleur Signification
Bleu Répertoire
Vert Fichier éxécutable
Bleu ciel Lien symbolique
Rose Fichier image
Rouge Fichier archive

Lister le contenu d'un dossier

Vous pouvez également indiquer quel dossier vous souhaitez lister de la manière suivante :

ls /var/www
  1. Affichez le contenu d'un dossier en affichant également les fichiers et dossiers cachés (commençant pas un point)
  2. Affichez le contenu d'un dossier en affichant également les droits, la taille, la date de création...
  3. Affichez le contenu d'un dossier en triant les fichiers et dossiers par date de modification
  4. Afficher le contenu d'un dossier en affichant la taille des éléments en Ko, Mo ou Go

Se déplacer

Pour vous déplacer dans l'arborescence de votre disque dur, vous pouvez utiliser la commande cd.

Vous pouvez également préciser le chemin vers lequel vous souhaitez vous déplacer :

cd /var/www

Si vous ne précisez pas de chemin, vous serez alors redirigé vers votre dossier home.

Le chemin que vous indiqué peut être en absolu (en commençant par le caractère "/") ou bien relatif (en indiquant directement le nom d'un dossier ou bien en commençant par les caractères "./").

cd .. vous permet de remonter dans le dossier parent.

Utilisez l'autocomplétion avec la touche Tab

Effectuer une recherche

La commande grep vous permet d'effectuer une recherche dans un fichier ou un dossier :

grep -r "texte recherché" ~/Downloads
