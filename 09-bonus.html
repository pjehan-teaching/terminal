Bonus

Exécuter à nouveau la dernière commande

La commande !! permet de re-exéctuer la dernière commande.

Cela peut-être pratique si vous devez exécuter une commande en super utilisateur :

sudo !!

Exécuter plusieurs commandes

Vous pouvez exécuter plusieurs commandes en les séparant par &&

git add . && git commit -m "message" && git push origin master
La commande précédente doit avoir fonctionnée pour que la suivante soit exécutée

Ouvrir un fichier de log

Il est souvent utile d'observer la fin de fichier de log afin de voir si des erreurs se produisent.

Pour cela, vous pouvez utiliser la commande tail qui permet d'afficher la fin d'un fichier en mode "watch" :

tail -f /var/log/apache2/error.log

Localiser un programme

Pour savoir où se trouve le fichier correspondant à une commande, vous pouvez utiliser le commande which :

which php
which which

Retoucher une image

Le programme imagemagick permet d'effectuer des retouches d'images en ligne de commande.

Il est par exemple possible de supprimer les espaces blancs autour d'une image :

convert input.png -trim output.png

Réduire la qualité d'une image :

convert input.png -quality 75 output.png

Ou encore "merger" des fichiers PDF :

convert pdf1.pdf pdf2.pdf output_pdf.pdf

Modifier des fichiers audio et vidéo

Le programme ffmpeg permet de modifier des fichiers audio en ligne de commande.

Il est par exemple possible de convertir le format d'un fichier :

ffmpeg -i input.mp4 output.avi

D'extraire l'audio d'une vidéo :

ffmpeg -i input.mp4 -vn -ab 320 output.mp3

Ou encore "merger" des fichiers vidéo :

ffmpeg -i "concat:input1.mp4|input2.mp4" -c copy output.mp4
