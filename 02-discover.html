Le shell

Un shell est une "coquille" positionnée autour d'un système d'expoitation (operating system ou OS). Il permet d'appeler des opérations qui seront ensuite exécutées par l'OS.

On peut distinguer 2 types de shell :

  • Les interfaces en ligne de commande (command-line interface ou CLI)
  • Les interfaces graphiques (graphical user interface ou GUI)

Le shell

Pour utiliser le CLI, il est nécessaire de connaitre les commandes disponibles ce qui représente une difficulté majeure pour la prise en main de l'outil.

Cependant, le CLI présente de nombreux avantages :

  • Plus rapide pour exécuter des actions
  • Plus de possibilités qu'avec une interface graphique
  • Automatisation des tâches
  • Indispensable pour administrer un serveur

Comparaison d'un copier / coller :

Windows VS MacOS VS Unix

Les systèmes d'exploitation MAC OS et Unix utilisent tous les deux le shell bash (Bourne Again SHell) tandis que Windows utilise cmd ou PowerShell.

Il existe également des alternatives telle que ZSH pour Linux

Bash VS CMD

Les commandes disponibles dans bash et cmd sont souvent différentes :

Action CMD Bash
Copier un fichier copy cp
Lister le contenu d'un dossier dir ls
Créer un dossier mkdir mkdir

De plus, bash et cmd n'utilisent pas la même syntaxe pour définir un chemin vers un fichier :

CMD Bash
cd \Users\johndoe cd /home/johndoe

Le terminal

Lorsque vous démarrez l'application terminal, une invite de commande s'affiche alors avec une ligne comme celle-ci :

username@computer-name:~$
  • username : Nom de l'utilisateur en cours
  • computer-name : Nom de la machine
  • ~ : Position actuelle sur le disque dur

Vous pouvez saisir une commande après le signe $ et appuyer sur la touche "Entrée" pour l'exécuter.

Le caractère tilde "~" correspond au dossier "home" de l'utilisateur en cours

Forcer la fermeture d'un programme

Si vous souhaitez forcer la fermeture d'un programme lancé depuis le terminal, vous pouvez appuyer sur les touches Ctrl + c.
